Inserting Images

Images use the <img> tag. The tag <img> is self enclosing.

The image tag needs you to tell it what is the source of the image. Source in html is src and it is an attribute of the <img> tag, that is followed by the value and by the attribute alt. This provides a text alternative in case the user can't view the image.

Src specifies the URL (address) of the image. Alt gives a description of the image. Alltogether:

<img src="images/nameofimage.jpg" alt="Description of image"/>

4. Adding a caption to an image

The <figure> tag specifies self-contained content. Used for images mostly. Think of it as a box, inside the box we put the image and we put a caption.

For the caption we use <figcaption>

All together looks like a russian doll:

        
          <figure>

          <img src="images/07-insertimagefromfolder.jpg" alt="Trump meltdown. Time magazine cover by Edel Rodriguez"/>

          <figcaption>Trump meltdown. Time magazine cover.</figcaption>

          </figure>
